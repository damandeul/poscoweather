# POSCO 제철소 기상 모니터링 대시보드

광양제철소·포항제철소의 실시간 기상 상태와 주변 도로 CCTV를 한 화면에서 확인하는 대시보드입니다.

## 기능

- **실시간 기상** (2분 간격 자동 갱신, [Open-Meteo](https://open-meteo.com/) — API 키 불필요)
  - 기온·체감온도, 풍향·풍속·돌풍, 습도, 강수량, 해면기압, 운량, 시정, 일출·일몰
  - 강풍·호우·뇌우·폭염·시정불량 등 자체 판정 경고 배지
- **대기질**: PM2.5 / PM10 (한국 기준 4단계 등급), NO₂, SO₂, O₃, 자외선지수
- **해양 상태**: 파고, 파향, 파주기, 해수면 온도 (두 제철소 모두 임해 공장)
- **24시간 예보 차트** (기온·강수확률, 호버 툴팁) 및 일별 예보
- **주변 CCTV**: 국가교통정보센터(ITS) 오픈API — 제철소 반경 도로 CCTV를 가까운 순으로 정렬, HLS 실시간 스트리밍 재생

## 실행

```bash
npm install
npm run dev   # http://localhost:3000
```

## CCTV 연동 (선택, 무료)

1. [국가교통정보센터 오픈데이터](https://www.its.go.kr/opendata/)에서 회원가입 후 **오픈API 인증키** 신청 (즉시 발급)
2. `.env.local.example`을 `.env.local`로 복사하고 키 입력:
   ```
   ITS_API_KEY=발급받은키
   ```
3. 개발 서버 재시작

키가 없어도 기상 대시보드는 모두 동작하며, CCTV 패널에 설정 안내가 표시됩니다.

## 참고

- 빌드 출력 폴더가 `.next` 대신 `.next-out`으로 설정되어 있습니다 ([next.config.ts](next.config.ts)). 기존 `.next` 폴더가 OS 수준 파일 잠금으로 접근 불가해진 것을 우회한 설정으로, 재부팅 후 `.next` 폴더를 삭제할 수 있다면 원복해도 됩니다.
- CCTV 스트림 URL은 ITS 서버에서 직접 제공되므로, HTTPS로 배포할 경우 일부 HTTP 스트림은 브라우저의 혼합 콘텐츠 정책으로 재생이 차단될 수 있습니다.
